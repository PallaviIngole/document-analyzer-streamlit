# Document Analyzer & Search Tool

A Python-based application that extracts text from PDF documents and enables fast keyword-based search and insights. Built with Streamlit for an interactive UI, this tool helps quickly analyze documents without using paid APIs.

---

## Features

* Upload any PDF document
* Extract clean text using pdfplumber
* Keyword search within the document
* Quick insights for Skills, Education, and Projects
* Simple, fast, and no API key required

---

## Tech Stack

* Python
* Streamlit
* pdfplumber
* Regex (re)

---

## Installation

```bash
pip install streamlit pdfplumber
```

---

## How to Run

```bash
streamlit run app.py
```

The app will open automatically in your browser.
